# Tenstorrent QuietBox RISC-V system

We have a single Tenstorrent QuietBox node with 4x Blackhole cards.

The host system is a single AMD 8124P 16-core processor with 512GB RAM.

To access this system and for usage instructions, please see the [COSMA support pages](https://cosma.readthedocs.io/en/latest/tenstorrent.md)
